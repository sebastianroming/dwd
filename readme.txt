Zeigt aktuelle Wetterwarnungen des Deutschen Wetterdienstes (DWD)

== Description ==

Ein Widget, welches ohne Konfigurationsaufwand eine Grafik mit Unwetterwarnungen für bestimmte Bundesländer (oder ganz Deutschland) einbindet.

== Installation ==

Installation via [WordPress Pluginverzeichnis](https://wordpress.org/plugins/dwd/).
